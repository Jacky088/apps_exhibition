# Apps Exhibition

> 一个用来快速搭建「APP 商店」风格展示页的 WordPress 插件

通过一个简单的短代码，你就可以在任意页面输出整洁的 APP 列表，适合个人/团队/公司以网页形式展示自家应用。

---

## 功能特性

- ✅ 在一个页面集中展示多个 APP 信息
- ✅ 支持显示应用名称、简介、下载按钮等信息
- ✅ 使用短代码一键插入到任意页面或文章
- ✅ 前端展示样式简洁，方便二次定制
- ✅ 基于 WordPress 插件机制，安装开箱即用

---

## 环境要求

- WordPress（建议 5.x 或更高版本）
- PHP 7.0+（建议 7.2 及以上）
- 具备安装插件的后台管理权限

> 以上为通用建议配置，具体要求以实际运行环境为准。

---

## 安装

### 方式一：通过 WordPress 后台上传（推荐）

1. 从本仓库下载插件压缩包（或在 **Releases** 中获取打包好的版本）。
2. 登录 WordPress 后台。
3. 打开 **「插件」 → 「安装插件」 → 「上传插件」**。
4. 选择下载好的压缩包并上传。
5. 安装完成后，点击 **「启用」** 插件。

### 方式二：手动上传到服务器

1. 将插件代码目录 `apps_exhibition` 上传到服务器的  
   `wp-content/plugins/` 目录下。
2. 登录 WordPress 后台。
3. 在 **「插件」** 列表中找到 **Apps Exhibition**。
4. 点击 **「启用」**。

---

## 快速开始

1. 登录 WordPress 后台，打开 **「页面 → 新建页面」**。
2. 为页面起一个名称，例如：`应用商店`。
3. 在页面内容编辑器中插入以下短代码：

   ```shortcode
   [apps_exhibition]

## 开发与自定义

如果你想在原有基础上进行二次开发或定制样式，可以参考以下方向：

在 assets/ 中修改或新增 CSS/JS，实现自定义样式和交互；

在 templates/ 中调整前端结构，增减展示字段或修改布局；

在 includes/ 中编写新的功能模块，例如：

从数据库或自定义文章类型中读取 APP 数据；

增加后台配置页面，支持可视化管理 APP 列表；

在 apps-exhibition.php 中注册更多短代码或挂载 WordPress 钩子。

建议在修改前先创建分支或备份代码，以便回滚。
