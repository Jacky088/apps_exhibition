# Apps Exhibition

一个用来快速搭建「APP 商店」风格展示页的 WordPress 插件。

通过一个简单的短代码，你就可以在任意页面输出整洁的 APP 列表，适合个人 / 团队 / 公司以网页形式展示自家应用。

---

## 功能特性

- ✅ 在一个页面集中展示多个 APP 信息  
- ✅ 支持显示应用名称、简介、下载按钮等信息  
- ✅ 使用短代码一键插入到任意页面或文章  
- ✅ 前端展示样式简洁，方便二次定制  
- ✅ 基于 WordPress 插件机制，安装即可使用  

---

## 环境要求

- WordPress（建议 5.x 或更高版本）  
- PHP 7.0+（建议 7.2 及以上）  
- 具备安装插件的后台管理权限  

---

## 安装教程

### 方法一：通过 WordPress 后台上传（推荐）

1. 从 GitHub 下载插件 ZIP 压缩包  
2. 登录 WordPress 后台  
3. 前往 **插件 → 安装插件 → 上传插件**  
3. 上传 ZIP 文件并安装  
4. 点击 **启用插件**  

---

### 方法二：手动上传到服务器

将 `apps_exhibition` 文件夹上传到以下路径：

```
wp-content/plugins/
```

然后：

1. 登录 WordPress 后台  
2. 前往 **插件** 页面  
3. 找到 **Apps Exhibition** 并点击 **启用**  

---

## 快速开始

1. 新建一个 WordPress 页面（例如：`应用商店`）  
2. 在页面内容中插入短代码：

```
[apps_exhibition]
```

3. 发布页面  
4. 前往该页面即可查看 APP 展示效果  

---

## 短代码说明

当前插件使用一个短代码：

```
[apps_exhibition]
```

- 可插入任意页面或文章  
- 如需增加参数（如分类过滤、显示数量等），可在源码中扩展逻辑

## 项目结构

```text

├─ apps-exhibition.php              # 插件入口文件，注册插件、挂载钩子、注册短代码等
```

## 开发与自定义

若你想扩展或定制，可以参考以下方向：

- 修改 `assets/` 中的 CSS / JS，实现自定义样式与交互；  
- 调整 `templates/` 中的前端结构，增减展示字段或修改布局；  
- 在 `includes/` 中开发新模块，例如：  
  - 从数据库或自定义文章类型读取 APP 数据；  
  - 添加后台可视化界面，管理 APP 列表；  
- 在 `apps-exhibition.php` 中注册更多短代码或 WordPress 钩子。  

> 建议修改前创建分支或备份代码，便于回滚。
